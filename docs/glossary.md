# Glossary

Unsloth: efficient LLM fine tuning.

LoRA: adapter based fine tuning.

QLoRA: LoRA with quantized model loading.

GGUF: local model format.

Ollama: local model runner.

Adapter: small trained weights added to a base model.

Dataset: examples used for training or evaluation.

Evaluation: repeatable model output checks.

Quantization: smaller numeric format to reduce size.
